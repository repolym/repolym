# راهنمای کامل نصب — سامانه المپیاد

## مشکل رفع‌شده در این نسخه
خطای "Account created but profile setup failed" برطرف شد.
علت: کد قدیمی سعی می‌کرد دستی پروفایل بسازد، اما Supabase trigger قبلاً این کار را کرده بود → conflict!
در نسخه جدید، همه‌چیز از طریق trigger دیتابیس انجام می‌شود.

---

## مرحله ۱ — Supabase (رایگان، بدون تحریم)

۱. به [supabase.com](https://supabase.com) بروید و با ایمیل ثبت‌نام کنید

۲. روی **New Project** کلیک کنید:
   - Name: `olympiad`
   - Database Password: یک رمز قوی بنویسید
   - Region: **West EU (Ireland)**

۳. بعد از ساخت، به **SQL Editor** بروید:
   - فایل `supabase-schema.sql` را باز کنید
   - همه محتوا را کپی کنید، paste کنید، **Run** کنید
   - پیام `Success` باید ظاهر شود ✅

۴. به **Settings → API** بروید و این دو مقدار را کپی کنید:
   - `Project URL` → مثال: `https://xxxx.supabase.co`
   - `anon public key` → یک کلید بلند

---

## مرحله ۲ — آماده‌سازی فایل‌ها

۱. زیپ را Extract کنید

۲. داخل پوشه `olympiad-app`، فایلی با نام `.env.local` بسازید (با Notepad):
```
VITE_SUPABASE_URL=آدرس_شما_از_Supabase
VITE_SUPABASE_ANON_KEY=کلید_anon_شما
```

۳. [Node.js](https://nodejs.org) را نصب کنید (نسخه LTS)

۴. Terminal/CMD باز کنید:
```bash
cd olympiad-app
npm install
npm run build
```
پوشه `dist` ساخته می‌شود ✅

---

## مرحله ۳ — Netlify (هاست رایگان، بدون تحریم)

۱. به [netlify.com](https://netlify.com) بروید و ثبت‌نام کنید (با GitHub یا ایمیل)

۲. روی **Add new site → Deploy manually** کلیک کنید

۳. پوشه `dist` را drag & drop کنید روی صفحه Netlify

۴. بعد از آپلود، به **Site configuration → Environment variables** بروید:
   - اضافه کنید: `VITE_SUPABASE_URL`
   - اضافه کنید: `VITE_SUPABASE_ANON_KEY`

۵. به **Deploys → Trigger deploy** کلیک کنید

۶. در Supabase → **Authentication → URL Configuration**:
   - Site URL: آدرس Netlify خود را وارد کنید
   - Redirect URLs: آدرس Netlify + `/**`

## اگر مشکل داشتید

در Supabase → **Authentication → Users** بررسی کنید آیا کاربر ساخته شده.
در Supabase → **Table Editor → users** بررسی کنید آیا پروفایل هم هست.
اگر کاربر هست اما پروفایل نیست → SQL Editor را دوباره اجرا کنید.
